Definição de algoritmo: sequência lógica para resolver determinados problemas.
Entrada: informações que foram recebidas. 
Processamento: o que é realizado com as informações recebidas. 
Saída: resultado da aplicação das ações.
Exemplo: fazer café                   
1. colocar água na chaleira
2. Ferver a água da chaleira
3. Colocar pó de café no filtro
4. despejar a água quente no pó de café
5. servir o café para beber
Palavras-chave da aula:
Algoritmo: sequência para resolver determinado problema
Entrada: informações recebidas
Processamento: etapa onde os passos descritos nas informações são aplicados
Saída: resultado final
Lógica: forma de organização para a solução de problemas
